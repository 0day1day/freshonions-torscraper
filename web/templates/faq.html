FAQ Question Answer Back
What do the colours of the onion links mean? Green means the site is live. Orange means we've been having some trouble connecting to it recently, but it may still be fine. Red means we haven't been able to connect to it for over four hours, and the link is likely to be dead.

How often are the links checked? Once an hour-ish.

Where do you get new onions from? This site runs a tor web spider that is constantly crawling every link in the database 24/7. New links are added as they are discovered.

What does "SHOW ALL" do? Certain onion links (particularly the ones entitled "Site Hosted by Freedom Hosting II") never seem to have any content and were cluttering up the interface. "SHOW ALL" reveals them. They are still listed in the JSON output, because if you can parse JSON then you can filter them out yourself (if that's what you want).

What do the [FAKE] and [GENUINE] tags mean? Sites listed in the /r/darknetmarkets superlist are tagged as genuine. Clones of these sites are tagged as fake. Note that clone detection is pretty primitive.
