What do the colours of the onion links mean? Green means the site is live. Orange means we've been having some trouble connecting to it recently, but it may still be fine. Red means we haven't been able to connect to it for over four hours, and the link is likely to be dead.

How often are the links checked? Sites that are up get checked every 1-4 hours. Sites that are dead get checked (60+rand(180))*(1.5^dead_in_a_row) where "dead_in_a_row" is the number of times this site has been checked and been found dead in a row. If a site is found to be up this is reset to zero.

Where do you get new onions from? This site runs a tor web spider that is constantly crawling every link in the database 24/7. New links are added as they are discovered.

What do the [F] and [G] tags mean? Sites listed in the /r/darknetmarkets superlist are tagged as [G]enuine. Clones of these sites are tagged as [F]ake. Note that clone detection is pretty primitive.
